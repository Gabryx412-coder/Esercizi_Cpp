# Calcolatrice C++ (Base)


**Descrizione**


Semplice calcolatrice in C++ che supporta le operazioni base: **somma**, **sottrazione**, **moltiplicazione** e **divisione**.


**File forniti**


- `calcolatrice.cpp` - codice sorgente principale.

**Note**


- Il programma gestisce l'input non numerico richiedendo nuovamente il valore.
- La divisione per zero è intercettata e gestita con un messaggio di errore.
- La precisione di stampa è impostata a 6 decimali, modificabile in `main` con `cout.precision()`.


**Miglioramenti possibili**


- Implementare supporto per operazioni su più numeri (es. somma di una lista).
- Aggiungere salvataggio della cronologia delle operazioni su file.
- Creare una versione con interfaccia grafica (es. con Qt).
